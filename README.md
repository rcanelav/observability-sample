Boilerplate for observability with opentelemetry, prometheus, grafana, loki, and jaeger.


## 1. Create virtual environment
    
    ```
    $ python3 -m venv venv
    ```

## 2. Activate virtual environment

    Linux
    ```
    $ source venv/bin/activate
    ```

    Windows
    ```
    $ venv\Scripts\activate
    ```

## 3. Check makefile or:

    ```
    $ make
    ```
